### 等比数列求和公式
    * Sn = a1*(1-q^n)/(1-q)=(a1-an*q)/(1-q)
### 等差数列求和公式
    * Sn = n*(a1+an)/2=a1*n+n*(n-1)/d
### 基姆拉尔森计算公式
    * W= (d+2*m+3*(m+1)/5+y+y/4-y/100+y/400+1)%7
    * W表示星期，W的取值范围是0~6，分别代表星期一到星期日。
    * 把每年的一月和二月当成前一年的13月和14月
    * w==0时是周日，w==1~6时是周一到周六
### 海伦公式
    * S=sqrt(p*(p-a)*(p-b)*(p-c))
    * p=(a+b+c)/2
### 模运算的分配律
    * (a + b) % p = (a % p + b % p) % p 
    * (a - b) % p = (a % p - b % p) % p 
    * (a * b) % p = (a % p * b % p) % p 
    * (a^b) % p = ((a % p)^b) % p
